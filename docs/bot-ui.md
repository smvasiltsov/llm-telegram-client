# Управление ботом через Telegram

## Основной вход
Все настройки выполняются в личных сообщениях с ботом через команду `groups`.

## Группы
- `groups` → выбор группы → список ролей.

## Роли
В карточке роли доступны:
- системный промпт;
- инструкция к сообщениям;
- инструкция для реплаев;
- LLM‑модель;
- переименование роли;
- сброс сессии;
- удаление роли.

## Создание роли
При создании роли бот последовательно спрашивает:
1) имя роли;
2) описание;
3) системный промпт (можно пропустить);
4) выбор LLM‑модели.

## Как бот отвечает
- В группах бот реагирует только на сообщения владельца.
- Команды работают только в личке с ботом.
