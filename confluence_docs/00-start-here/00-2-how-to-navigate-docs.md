## Source-of-Truth Principles
- Repository documentation is the primary technical source.
- Confluence pages mirror structured, stakeholder-friendly views.
- Architectural decisions must be documented in ADR pages.
- Operational rules and checklists live under `60.*`.

## How to Keep Navigation Consistent
- Keep numbered prefixes in page titles.
- Keep one topic per page.
- Move deprecated material to `90. Archive` instead of deleting history.
- Cross-link related pages where a reader needs jump context.
